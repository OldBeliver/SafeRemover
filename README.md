# SafeRemover

**SafeRemover** — это интерактивная консольная утилита на C#, предназначенная для удобного и безопасного извлечения USB-накопителей (флешек) в Windows. Программа автоматически находит подключенные съемные устройства и выводит подробную информацию об их объеме.

## 🚀 Основные возможности

* **Автоматическое обнаружение:** Программа сама находит все готовые к работе съемные диски.
* **Информативность:** Отображение занятого, свободного и общего объема памяти для каждого накопителя.
* **Интерактивное меню:** Удобный выбор диска для извлечения по номеру или возможность извлечь все устройства сразу.
* **Горячие клавиши:**
    * `F5` — мгновенное обновление списка подключенных дисков.
    * `Esc` — быстрый выход из программы.
* **Безопасность:** Использует низкоуровневые команды Windows для корректного завершения работы с устройством.

## 🛠 Технологический стек

* **Язык:** C#
* **Платформа:** .NET Framework / .NET (Windows)
* **API:** Windows API (kernel32.dll: CreateFile, DeviceIoControl)

## 📋 Как это работает

Для извлечения накопителя программа выполняет следующие шаги через WinAPI:
1. Получает список всех съемных устройств через `DriveInfo`.
2. По запросу пользователя открывает дескриптор устройства с помощью `CreateFile`.
3. Отправляет управляющий код `IOCTL_STORAGE_EJECT_MEDIA` (0x2D4808) через `DeviceIoControl`, что инициирует безопасное извлечение.
4. Закрывает дескриптор и освобождает ресурсы.

## 💻 Использование

Просто запустите файл `SafeRemover.exe`. Программа перейдет в режим ожидания команд.

### Пример интерфейса:
```text
Обнаружены следующие флеш-диски:
1: E:\ (Занято: 1.45 ГБ, Свободно: 13.55 ГБ, Всего: 15.00 ГБ)
2: F:\ (Занято: 500.00 МБ, Свободно: 3.50 ГБ, Всего: 4.00 ГБ)

Введите номер диска для извлечения или 0 для всех.

[F5] - Обновить список | [Esc] - Выход
```
