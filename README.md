# SafeRemover

**SafeRemover** — это компактная консольная утилита на C#, предназначенная для программного и безопасного извлечения USB-накопителей (флешек) в операционной системе Windows. 

Программа имитирует стандартную функцию Windows «Безопасное извлечение устройства», гарантируя, что все операции записи завершены, а кэш данных сброшен на диск перед его отключением.

## 🚀 Основные возможности

* **Программное извлечение:** Позволяет отключать накопители через командную строку или скрипты.
* **Безопасность данных:** Перед извлечением программа пытается принудительно размонтировать том и заблокировать его, что предотвращает повреждение файловой системы.
* **Работа через WinAPI:** Прямое взаимодействие с низкоуровневыми функциями Windows (`setupapi.dll`, `cfgmgr32.dll`).
* **Минимализм:** Один исполняемый файл, отсутствие лишних зависимостей.

## 🛠 Технологический стек

* **Язык:** C#
* **Платформа:** .NET Framework / .NET (Windows)
* **API:** Windows SetupAPI, Configuration Manager API

## 📋 Как это работает

Программа выполняет последовательность действий для корректного отключения:
1.  Открывает дескриптор (handle) указанного диска.
2.  Отправляет команду на блокировку тома (`FSCTL_LOCK_VOLUME`).
3.  Размонтирует файловую систему (`FSCTL_DISMOUNT_VOLUME`).
4.  Отправляет запрос на программное извлечение физического устройства (`CM_Request_Device_Eject`).

## 💻 Использование

Запустите скомпилированный файл через терминал (cmd/PowerShell), передав букву диска в качестве аргумента:

```bash
SafeRemover.exe E:
```

### Пример вывода:
```text
Поиск устройства для диска E:...
Попытка блокировки тома...
Устройство 'USB Mass Storage Device' успешно извлечено.
```
